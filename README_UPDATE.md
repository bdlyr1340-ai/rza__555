# تحديث v7 — العداد كل ثانية + سرعة قصوى

  ## الملفان
  1. `bot/handlers/start.py`  ← العداد يتحرّك كل ثانية الآن
  2. `bot/services/sheerid.py` ← أسرع 4× من الأصل

  ## ما الجديد

  ### ⏱ العداد
  - يحدّث **كل ثانية** (00:00 → 00:01 → 00:02 → 00:03 ...)
  - بنفس الشكل الذي طلبته:
    - 🔄 = الخطوة الحالية
    - ✅ = خطوة منتهية
    - ⬜ = خطوة قادمة
    - ❌ = فشل

  ### ⚡ السرعة
  - جميع الانتظارات الإنسانية مقطوعة لـ **25%** من الأصل
  - جميع انتظارات الصفحات مقطوعة لـ **35%** من الأصل
  - النتيجة: تسجيل الدخول أسرع **3-4 أضعاف**

  ### 📸 لقطة شاشة عند رفض SheerID
  - (من v6) عندما يرفض SheerID، يرسل لك صورة + JSON

  ## كيف ترفع
  1. حمّل الملف، استخرج الـ zip
  2. ارفع `start.py` إلى `bot/handlers/` على GitHub (Edit → استبدل المحتوى → Commit)
  3. ارفع `sheerid.py` إلى `bot/services/` بنفس الطريقة
  4. انتظر دقيقتين حتى Railway يعمل Redeploy
  5. جرّب البوت

  ## ضبط السرعة (اختياري)
  لو أردت أسرع، أضف في Railway → Variables:
  ```
  SHEERID_SPEED_FACTOR=0.15
  ```
  لو ظهرت مشاكل كشف، ارجع للسرعة الأصلية:
  ```
  SHEERID_SPEED_FACTOR=1.0
  ```
